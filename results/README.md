Transcription-associated DNA break mapping analysis

This repository contains representative R analysis workflows related to genome-wide mapping of endogenous transcription-associated DNA double-strand breaks (DSBs) in human cells.

The code focuses on gene-level integration of endogenous DSBs with transcription stress-associated genomic features, including R-loops, TOP1 occupancy, and TOP1 cleavage complexes (TOP1cc). These analyses were developed as part of my PhD work investigating how superenhancer-driven transcription shapes genome fragility in cancer.

Overview

The main workflow scores genes for transcription-associated DNA break stress using four genome-wide features:

* endogenous DSB density from sBLISS
* R-loop density from DRIP-seq
* TOP1 occupancy
* TOP1cc signal

For each feature, gene-level signal density is converted into an empirical P value based on rank. Higher signal values correspond to lower empirical P values. These P values are then transformed into Z-scores and combined using an unweighted Liptak/Stouffer Z-score approach.

The final transcription-stress score is used to rank genes and test enrichment of biologically relevant gene sets, including superenhancer-regulated genes, oncogenes, and highly expressed genes.

Scripts

01_score_transcription_stress_genes.R

This script reads processed gene-level signal tables for sBLISS, DRIP-seq, TOP1, and TOP1cc. It standardizes gene names, ranks genes within each dataset, converts ranks to empirical P values, transforms P values into Z-scores, and combines the four features into a transcription-stress score.

Genes not among the top-ranked genes in one or more parameters are excluded before final scoring.

02_gene_set_enrichment.R

This script tests whether defined gene sets are enriched among high-scoring transcription-stress genes using the hypergeometric test.

Gene sets include examples such as:

* superenhancer-regulated genes
* oncogenes
* superenhancer-regulated oncogenes
* highly expressed genes
* highly expressed superenhancer-regulated genes

03_plot_transcription_stress_results.R

This script generates example plots from the scoring and enrichment results, including:

* ranked transcription-stress gene bar plots
* heatmaps of individual feature Z-scores
* gene-set enrichment plots

Example data

The data/example/ folder contains small example input files that demonstrate the expected data format. These are not full genome-wide datasets and are included only to make the workflow structure clear.

Full sequencing-derived tables are not included in this repository because they are large processed outputs generated from genome-wide datasets.

Input data

The full analyses were based on processed gene-level signal tables derived from genome-wide datasets, including sBLISS, DRIP-seq, TOP1 ChIP-seq, and TOP1cc datasets.

The DSB signal used in the example workflow is SE_mean, representing the control condition originally generated from scramble siRNA and empty-vector control samples.

How to run

Run the scripts in order from the main repository directory:

source("R/01_score_transcription_stress_genes.R")
source("R/02_gene_set_enrichment.R")
source("R/03_plot_transcription_stress_results.R")

The scripts will write output tables to results/ and plots to figures/.

Notes

This repository is intended to provide a clean and readable example of the analysis logic used for transcription-associated DSB scoring and gene-set enrichment. It is not a full raw-data processing pipeline and does not include large sequencing files such as FASTQ, BAM, bigWig, or full genome-wide intermediate files.
